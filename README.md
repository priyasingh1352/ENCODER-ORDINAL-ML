# Ordinal-Encoding-sklearn-Map-ML
# 📌 Description
 A hands-on Python project demonstrating Ordinal Encoding to convert categorical data into ordered numerical values for machine learning models using Scikit-learn and Python map() function.

# 📌 What is Ordinal Encoding?
Ordinal Encoding is a technique used to convert categorical data into numerical values while keeping the correct order or ranking of the categories.

# ⚙️ Workflow
* Import required libraries (pandas, scikit-learn)
* Create dataset
* Define category order (S < M < L < XL < XXL)
* Apply OrdinalEncoder from Scikit-learn
* Perform manual encoding using map() function
* Transform categorical data into numeric format
# 🚀 Features
* Handles ordered categorical data efficiently
* Uses both automatic (OrdinalEncoder) and manual (map) methods
* Preserves category ranking
* Simple and beginner-friendly implementation
* Useful for machine learning preprocessing
# 🛠️ Technologies Used
* Python
* Pandas
* Scikit-learn (OrdinalEncoder)
* Jupyter Notebook
